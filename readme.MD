# D3 starter kit for Mr. Hiekkaranta

# Requirements
* git
* node & npm
* heroku tool for deployment

## To install
`git clone xxx`
`cd xxx`
`npm install`

## To start development environment
`npm run start`

## To build static assets for production
`npm run build`

## To set up heroku for deployment
`heroku create app`

## To deploy to heroku
`git push heroku`
